# 🚀 Syntax Sphere

**Learn Programming Languages Faster, Smarter, and Better.**

Syntax Sphere is an AI-powered learning platform designed to help students, beginners, and developers master programming languages through interactive learning, real-world examples, AI assistance, quizzes, and hands-on coding practice.

---

## 📖 About the Project

Learning a new programming language can be overwhelming due to complex syntax, scattered resources, and lack of personalized guidance.

**Syntax Sphere** solves this problem by providing an intelligent learning environment where users can:

* 📚 Learn multiple programming languages in one place
* 🤖 Get AI-powered explanations for difficult concepts
* 💻 Practice with coding exercises and challenges
* 🧠 Understand syntax through real-world examples
* ⚡ Learn faster with structured roadmaps
* 🎯 Track progress and improve consistently

Whether you're starting your first programming language or expanding your skills, Syntax Sphere helps you learn efficiently and confidently.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/<username>/syntax_sphere.git
```

### Navigate to the Project

```bash
cd syntax_sphere
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
